# Capturer
Linux上的截图/录屏/Gif软件。

## Shortcut
### 截图
 - `F1` : 截图
 - `W / A / S / D` : (逐像素)移动选中区域
 - `Ctrl + UP/DOWN/LEFT/RIGHT` : (逐像素)扩大边界
 - `Shift + UP/DOWN/LEFT/RIGHT` : (逐像)缩小边界

### Video
 - `Ctrl+Alt+V` : 开始
 - `Ctrl+Alt+V` : 结束

### GIF
 - `Ctrl+Alt+G` : 开始
 - `Ctrl+Alt+G` : 结束

### 其他
 - `ESC` : 退出(截图/贴图/GIF/录屏/...)
