# Capturer
Linux上的截图/录屏/Gif软件。


## Todo
- [x] 截图 选择: F1, 退出: Esc, 确认: Enter, 将最后一个截图钉在桌面: F3
- [x] 录屏 开始: Ctrl + Alt + V, 退出: Esc, 结束: Ctrl + Alt + V
- [x] GIF 开始: Ctrl + Alt + G, 退出: Esc, 结束: Ctrl + Alt + G


## 依赖
- Qt5
- FFmpeg
